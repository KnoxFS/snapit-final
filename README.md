## Getting Started

> This project uses **yarn** as primary package manager. If you want to use it, run the following command: `npm i -g yarn`


First, install all the necessary packages.

```bash
npm install
# or
yarn
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Env vars

Check the *.env.example* file to see which keys you'll have to insert into the *.env.local* file (create this file if doesn't exist).
